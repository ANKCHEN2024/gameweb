Patch 2.x rebuilt Cyberpunk 2077's perk trees. These builds match the current meta for new and returning players.

## S-tier build archetypes

| Build | Main attribute | Playstyle |
| --- | --- | --- |
| Netrunner hacker | Intelligence | Control fights before shooting |
| Stealth pistol | Cool + Reflexes | Headshot assassin |
| Sandevistan slasher | Reflexes | High-speed melee |
| Shotgun bruiser | Body + Street Brawler | Close-range tank |

## Netrunner starter path

1. Raise **Intelligence** first
2. Grab quickhack damage and RAM upgrades
3. Use cameras and ping to clear rooms remotely
4. Keep one pistol for emergencies

**Best for:** players who like planning and crowd control

## Stealth pistol path

1. Prioritize **Cool** for headshot multipliers
2. Add silenced pistols and crit perks
3. Use stealth movement perks before engaging
4. Take cover between picks

**Best for:** immersive stealth runs

## Sandevistan combat path

1. Stack **Reflexes** and Sandevistan cyberware
2. Add melee or SMG perks for time-dilated burst
3. Invest in movement cyberware for gap close

**Best for:** action players who dislike hacking menus

## Common build mistakes

- Spreading attributes evenly
- Ignoring armor and resistances on cyberware
- Buying every weapon instead of mastering one type
- Skipping vehicle/combat quickhack utilities

## Bottom line

Pick one primary attribute line and commit through midgame. Netrunner and Sandevistan remain the strongest Patch 2.x templates for most players.
