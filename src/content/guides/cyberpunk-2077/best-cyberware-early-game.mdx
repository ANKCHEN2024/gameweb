Cyberware defines your build in Cyberpunk 2077. Early purchases should match your attribute plan — not random legendary drops.

## Early cyberware priorities by build

| Build | Buy first |
| --- | --- |
| Netrunner | RAM boost, upload speed, deck perks |
| Stealth | Optical cloaking variants, crit cyberware |
| Sandevistan | Sandevistan OS, reflex boosters |
| Tank | Armor subdermal, health regen mods |

## Universal strong picks

- **Kerenzikov** (if reflex build) for dodge shooting
- **RAM/fragment upgrades** for quickhack chains
- **Armor plating** when capacity allows
- **Smart link** only if using smart weapons

## Ripperdoc shopping rules

1. Check all districts — stock rotates
2. Buy capacity-related upgrades before luxury chrome
3. Do not install conflicting OS types
4. Read perk requirements before purchase

## Capacity management

Patch 2.x ties capacity to attributes and gear perks. If you are over budget:

- Remove low-impact mods first
- Upgrade clothing with capacity perks
- Respec only after planning full loadout

## Mistakes to avoid

- Installing Sandevistan on a netrunner with no reflex investment
- Ignoring armor for weapon cyberware
- Buying legendary gear you cannot activate yet

## Bottom line

Match cyberware to your main attribute line. Netrunners buy RAM, fighters buy OS and armor, stealth buys crit and mobility tools — never the other way around.
