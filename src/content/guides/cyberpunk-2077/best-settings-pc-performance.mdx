Cyberpunk 2077 looks incredible but can crush unprepared PCs. These settings balance clarity and frame rate after Patch 2.x.

## Quick presets

| GPU tier | Target |
| --- | --- |
| RTX 4060 / RX 7700 XT | 1440p DLSS/FSR Quality, RT off |
| RTX 3070 class | 1440p DLSS Balanced, limited RT |
| RTX 4080+ | 4K DLSS Quality, RT medium+ |

## Settings to lower first

1. **Crowd density** — big CPU impact in city zones
2. **Ray tracing** — massive GPU hit
3. **Volume fog / screen space reflections** — moderate hit
4. **Shadows** — high cost for minor gameplay benefit

## Settings worth keeping high

- Texture quality (if VRAM allows)
- Anisotropic filtering
- Character detail in cutscenes if GPU headroom exists

## Upscaling guide

- **DLSS** on NVIDIA — start Quality, drop to Balanced if needed
- **FSR 2/3** on AMD — similar tier approach
- **Frame Generation** — great at 1440p+ if you accept latency tradeoff

## CPU tips

- Close background browser tabs
- Use fullscreen borderless carefully — fullscreen exclusive can help on some systems
- Shader compilation stutter improves after initial playtime

## Drive and OS

- Install on SSD
- Update GPU drivers before benchmarking
- Disable overlay apps if micro-stutter appears

## Bottom line

Turn off ray tracing on mid-range hardware, use DLSS/FSR Quality, and cut crowd density first. Cyberpunk is playable and gorgeous once upscaling is configured correctly.
