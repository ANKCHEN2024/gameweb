Meccha Chameleon is a PC-first party game, but control choice changes how you paint, scan, and reposition under pressure.

## Keyboard and mouse recommendations

| Setting | Suggested starting point |
| --- | --- |
| Mouse DPI | 800–1600 |
| In-game sensitivity | Low–medium |
| FOV | 90–100 (seeker), 80–90 (hider) |
| Paint button | Mouse side button if available |

**Pros:** faster camera snaps, precise seeker sweeps  
**Cons:** jittery paint strokes if sensitivity is too high

## Controller recommendations

| Setting | Suggested starting point |
| --- | --- |
| Look sensitivity | 45–55% |
| Move deadzone | 5–10% |
| Paint trigger | Right trigger with light pressure practice |
| Vibration | Off for cleaner audio cues |

**Pros:** smooth movement, comfortable couch sessions  
**Cons:** slower 180° seeker checks without claw grip

## Role-based setup

### Hider-focused

- Lower look sensitivity for stable paint alignment
- Toggle crouch if available to reduce accidental movement
- Use wired connection to avoid Bluetooth input lag

### Seeker-focused

- Slightly higher sensitivity for lane sweeps
- Higher FOV to catch edge silhouettes
- Bind ping/callout if supported in lobby modes

## PC performance basics

- Cap FPS only if your monitor is lower refresh; uncapped input can feel better on high-refresh displays
- Windowed fullscreen often reduces alt-tab stutter on Windows
- Update GPU drivers before judging sensitivity — stutter feels like bad aim

## Bottom line

There is no universal best input. Seekers lean keyboard/mouse; hiders often prefer controller for paint control. Pick one, tune sensitivity down until smooth, and stop changing settings mid-session.
