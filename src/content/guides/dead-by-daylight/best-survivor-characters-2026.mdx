Characters in Dead by Daylight are perk containers first, cosmetics second. Level the right roster to unlock flexible builds.

## Top survivors to level for perks

| Survivor | Teachable value |
| --- | --- |
| Bill | Borrowed Time meta |
| Meg | Sprint Burst / adrenaline paths |
| Claudette | Self-care and heal builds |
| David | Dead Hard access |
| Yun-Jin | Exhaustion alternatives |

## Solo queue recommendations

- **Claudette** — self-sufficient healing
- **Meg** — exhaustion perk access early
- **Bill** — save builds for random teammates

## SWF recommendations

- One dedicated healer / support build
- One exhaustion meta user
- One gen focus with Prove Thyself style perks

## Leveling order strategy

1. Unlock one exhaustion perk character
2. Unlock save perk (BT) character
3. Unlock information perk character
4. Main cosmetic favorite afterward

## Bottom line

Level survivors for teachables, not lore. Bill, David, Meg, and Claudette give the most build flexibility in 2026 solo queue.
