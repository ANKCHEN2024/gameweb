Looping is the core survivor skill in Dead by Daylight. Good loops buy time for generators; bad loops die on first chase.

## Strong loop ingredients

- At least one safe pallet
- A window vault with line-of-sight break
- Connection path to another loop (tile chain)

## Basic loop rules

1. **Run the loop direction that keeps the pallet between you and killer**
2. **Vault windows late** when killer respects the tile
3. **Fake vault** occasionally at intermediate MMR
4. **Leave the loop** before killer breaks pallet and re-angles

## Pallet discipline

| Situation | Action |
| --- | --- |
| Killer respects loop | Hold pallet, waste time |
| Killer commits through | Drop and transition |
| Killer breaks early | Run next tile without dropping |

## Window mind games

- Fast vault only when killer swings
- Slow vault bait can work once per chase
- Do not triple loop same window — predictable

## Exhaustion perk timing

- **Dead Hard:** use at window or to reach next pallet
- **Sprint Burst:** activate leaving gen or entering killer TR
- Never burn exhaustion perks for style points

## Practice drill

- Load custom game or PTB
- Chain two loops without taking damage
- Record chases and note where you dropped pallets too early

## Bottom line

Looping is time management. Respect tiles, delay pallet drops, and chain structures — three good chases win more games than one flashy dead hard clip.
