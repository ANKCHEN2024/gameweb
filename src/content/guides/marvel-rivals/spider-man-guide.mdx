Spider-Man is Marvel Rivals' flagship mobility Duelist. He wins on angle creation, not face-tanking damage.

## Core combo loop

1. Swing in from off-angle
2. Dive priority target with melee + abilities
3. Web out vertically
4. Re-enter when cooldowns reset

## Movement fundamentals

- **Always enter with height** — ground approaches get focused
- **Use environment corners** — break hitscan tracking
- **Save escape web** — never spend both mobility tools on engage

## Target priority

| Priority | Target |
| --- | --- |
| 1 | Low HP Strategists |
| 2 | Isolated Duelists |
| 3 | Cooldown-burned Vanguards |

## Team fight rules

- Follow tank engage — do not start fights alone
- Ping before dive so supports can follow
- Track enemy peel cooldowns (stuns, knockbacks)

## Practice drill

Custom match: traverse map using only webs without touching ground for 60 seconds. Builds muscle memory for escape routes.

## Bottom line

Spider-Man rewards disciplined dives. Swing in with a plan, burst a priority target, and web out before focus fire lands.
