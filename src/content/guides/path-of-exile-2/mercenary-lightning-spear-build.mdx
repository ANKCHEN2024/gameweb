Lightning Spear Mercenary is the community's favorite onboarding build for good reason — clear, range, and upgrade path.

## Core skill setup

- **Main:** Lightning Spear
- **Secondary:** Single-target spear skill for bosses
- **Movement:** Dash or mobility skill as unlocked
- **Aura/buff:** Damage or accuracy buff when mana allows

## Passive tree path

1. Life and resist wheels first
2. Projectile damage cluster
3. Attack speed or crit — pick one theme
4. Avoid travel nodes that do not hit notable passives

## Gear priorities

| Slot | Affix |
| --- | --- |
| Weapon | Physical + lightning damage |
| Body | Life + resists |
| Boots | Movement speed |
| Gloves | Attack speed |

## Boss rotation

1. Apply exposure or debuff if available
2. Burst with single-target skill during boss phase windows
3. Keep moving during telegraphed slams

## Bottom line

Stack life, link damage supports on Lightning Spear, and add a boss skill early. Mercenary Lightning Spear carries campaigns and maps with modest investment.
