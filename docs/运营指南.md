# SavePoint Guides 运营指南

本文档记录英文爆款游戏攻略站的选题、导航结构、变现与部署流程。

## 一、站点定位

**只做当下 Steam 爆款**，不做常青老游戏。当游戏热度下降后停更，转向下一款热门作。

## 二、导航结构（多游戏 Hub 模式）

游戏数量增加时，**顶栏不再逐个列出游戏**，避免导航爆炸。

```
顶栏：Home | Games | Guides | About | [搜索框]
         ↓                    ↓
    /games/ 聚合页      /guides/ 全攻略索引（按游戏筛选）
         ↓                    ↓
    /games/<slug>/       /guides/<slug>/ 攻略详情
    （左侧攻略目录）
```

### 关键文件

| 文件 | 作用 |
|------|------|
| [`src/components/Header.astro`](../src/components/Header.astro) | Home / Games / Guides / About + 搜索 |
| [`src/pages/games/index.astro`](../src/pages/games/index.astro) | 游戏聚合页（筛选） |
| [`src/pages/guides/index.astro`](../src/pages/guides/index.astro) | 全攻略索引（按游戏 Tab 筛选） |
| [`src/pages/games/[game].astro`](../src/pages/games/[game].astro) | 游戏专区 + 左侧攻略目录 |
| [`src/pages/search.astro`](../src/pages/search.astro) | Pagefind 站内搜索 |
| [`src/pages/index.astro`](../src/pages/index.astro) | 首页：Trending + 热门标签 + Latest |
| [`src/consts.ts`](../src/consts.ts) | 游戏元数据（含 trending 标记） |

### 首页原则

- 展示 **Trending Now**、**Popular topics**（热门标签）、**Latest Guides**
- 不在首页为每个游戏渲染 highlights 区块
- 全量游戏列表在 `/games/`，全量攻略在 `/guides/`

### 用户浏览路径

1. **找游戏** → `/games/` 或首页 Trending
2. **找攻略** → `/guides/` 按游戏 Tab 筛选，或顶栏搜索
3. **读某游戏全部攻略** → `/games/<slug>/` 左侧目录快速跳转
4. **读完一篇** → 底部「More guides」+ 相关推荐

## 三、每周爆款选题流程

### 1. 查 Steam Top Sellers（每周一）

- 打开 [Steam Weekly Top Sellers](https://store.steampowered.com/charts/topsellers)
- 记录前 20 名付费游戏（排除纯 F2P 可选）
- 标注：新作 / 夏促回流 / 长期热门

### 2. 评估是否上架

| 信号 | 动作 |
|------|------|
| 新进 Top 10 且发售 < 30 天 | 立即开写 5 篇攻略 |
| 连续 2 周 Top 20 | 列入观察，准备选题 |
| 连续 4 周不在 Top 50 | 停更，标记为非 trending |
| 有大版本更新 / DLC | 追更 2–3 篇补丁攻略 |

### 3. 选题公式

**游戏名 + 具体问题**，例如：

- `meccha chameleon best hiding spots`
- `path of exile 2 best starter build`
- `cyberpunk 2077 phantom liberty when to start`

### 4. 下架标准

- 游戏热度连续 4 周跌出 Steam Top 50
- 将 `trending: false` 写入 `consts.ts`
- 文章保留（仍有长尾流量），但不再新增

## 四、如何新增游戏

1. 在 [`src/consts.ts`](../src/consts.ts) 的 `GAMES` 中添加条目：

```ts
'game-slug': {
  title: 'Game Name',
  description: '...',
  steamUrl: '...',
  affiliateLabel: 'Buy ... on Steam',
  trending: true,
  trendingOrder: 5,
  releaseDate: '2026-01-01',
},
```

2. 在 [`src/content.config.ts`](../src/content.config.ts) 的 `gameSlugs` 枚举中加入 slug
3. 在 `src/content/guides/<game-slug>/` 下创建 MDX 文章
4. `npm run build` 验证

## 五、如何新增攻略

在对应游戏目录下新建 `.mdx`：

```yaml
---
title: "英文标题"
description: "meta 描述"
game: meccha-chameleon
publishedAt: 2026-07-10
tags: ["tag1"]
faq:
  - question: "问题？"
    answer: "回答。"
---
```

## 六、站内搜索（Pagefind）

构建命令会自动索引：

```powershell
npm run build
```

`postbuild` 脚本执行 `pagefind --site dist`，搜索页位于 `/search/`。

开发模式（`npm run dev`）无搜索索引，需 build 后 preview 测试。
